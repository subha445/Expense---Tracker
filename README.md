# 💰 Expense Tracker App

A simple and modern **Expense Tracker Web Application** built using HTML, CSS, and JavaScript.  
It helps users track income and expenses with real-time balance updates and visual charts.

---

## 🚀 Live Demo
👉 https://subha445.github.io/Expense---Tracker/

---

## Features

📈 Track Income and Expenses
💰 Real-Time Balance Updates
📝 Transaction History
🗑️ Delete Transactions
💾 LocalStorage Support
📱 Mobile-Friendly Design
🎨 Clean and Modern UI
⚡ Fast and Lightweight

---

## 🧠 How It Works

- User enters transaction details
- Data stored in JavaScript array
- UI updates dynamically using DOM manipulation
- Balance is calculated instantly
- Chart updates automatically
- Data persists using browser LocalStorage

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)
- Chart.js
